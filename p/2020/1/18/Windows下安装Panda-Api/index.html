Windows下安装Panda Api

下载Windows版的panda-api压缩包，解压搜到自己需要的目录

然后在命令行下进入panda-api文件夹目录

然后运行命令，就可以访问文档服务了。

.\panda-api.exe

如果遇到报错：

Error: Os {code: 10013, kind: PermissionDenied, message:"以一种访问权限不允许的方式做了一个访问套接字的尝试。"}

这个原因是默认端口9000被占用了

我们可以改动启动端口：

.\panda-api.exe -p 9001
