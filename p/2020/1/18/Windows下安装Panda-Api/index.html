Windows下安装Panda Api

到这个地址https://github.com/arlicle/panda-api/releases/latest 下载Windows版的panda-api安装包，解压搜到自己需要的目录 ，然后点击install.bat进行安装。

安装完成后，关闭终端，重新打开，就可以在自己的文档目录运行命令

panda -V

如果显示了panda aip的版本号，那么就是安装成功了。
